для получения свежей копии скриптов с GitHab надо в терминале, в папке со скриптами набрать
    git pull

    - если при обновлении выходят ошибки скачивания, вероятнее всего в файлах были сделаны изменения

    - проверить это можно командой 
        git status

    - если вывод команды будет содержать список файлов обозначенных красным или зелёным цветом, то надо надо сделать откат до предыдущей версии

    - в случае списка файлов красного цвета выполнить команду
        git restore *
        
    - в случае списка файлов зелёного цвета, что очень мало вероятно, придётся выполнить две команды
        git restore --staged *
        git restore *
