Тестовый запуск
 - Для тестового запуска работы скрпита запустить в терминале:
        python runtest.py (версия Linux - python3 runtest.py)

    * при нахождении годной для покупки монеты её данные записываются в файл symbol.json
    * при удачной покупке монеты, данные заказа записываются в файл BUY_order_receipt.py
    * при удачной продаже монеты, данные заказа записываются в файл SELL_order_receipt.py


Рабочий запуск
 - Для запуска работы скрипта выполнить команду в терминале:
        python main.py (версия Linux - python3 main.py)

    * при нахождении годной для покупки монеты её данные записываются в файл symbol.json
    * при удачной покупке монеты, данные заказа записываются в файл BUY_order_receipt.py
    * при удачной продаже монеты, данные заказа записываются в файл SELL_order_receipt.py

Вспомогательные и информационные скрипты (работают с обычными ключами API, не тестовыми)
    - скрипт отображения данных пользователя 
            python client.py (версия Linux - python3 client.py)
    
    - скрипт выбора монеты по максимальному процентному росту стоимости
            python coin.py (версия Linux - python3 coin.py)
